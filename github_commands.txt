# GitHub'a Proje Yükleme Komutları

# 1. Git yüklü değilse önce kurun: https://git-scm.com/download/win

# 2. PowerShell'de şu komutları sırayla çalıştırın:

# Git repository başlat
git init

# Tüm dosyaları ekle
git add .

# İlk commit
git commit -m "Initial commit: ITU Ders Kontenjan Bot

✨ Features:
- Multi-user course tracking
- Real-time quota notifications
- SQLite database
- Telegram bot commands
- Automated testing
- GitHub Actions CI/CD

🤖 Bot Commands:
- /start - Start bot
- /add_course - Add course
- /remove_course - Remove course
- /list_courses - List courses
- /status - Bot status

🛠️ Tech Stack:
- Python 3.8+
- Async/await
- SQLite
- Telegram Bot API
- GitHub Actions"

# Ana branch'i main olarak ayarla
git branch -M main

# GitHub repository'yi remote olarak ekle
# DEĞIŞTIR: yourusername yerine kendi GitHub kullanıcı adınızı yazın
git remote add origin https://github.com/yourusername/itu-ders-kontenjan-bot.git

# GitHub'a push et
git push -u origin main

# Başarılı! Repository artık GitHub'da
